Раздел 1. Введение в IT
IT — это область, занимающаяся обработкой, хранением, передачей и защитой информации.
Основные направления: Программирование, Сети, Базы данных, AI/ML, Кибербезопасность.
Как начать: Освойте базовые термины, операционную систему, затем переходите к коду.

Раздел 2. Компьютерное железо
CPU (Процессор) — мозг компьютера. Важны частота и ядра.
RAM (ОЗУ) — временная память. 16-32 ГБ нужно для видеомонтажа и игр.
GPU (Видеокарта) — важна для игр и обучения искусственного интеллекта (AI).

Раздел 5. Программирование (Рейтинг 2025)
1. Python: Лидер для AI, Data Science и веба.
2. C/C++: Системное ПО, игры, высокая производительность.
3. Java: Android, корпоративный софт.
4. JavaScript: Веб-разработка (Frontend/Backend).
5. Go/Rust: Серверы и безопасные системы.

Раздел 6. Базы данных
SQL (Реляционные): MySQL, PostgreSQL. Пример: SELECT * FROM users.
NoSQL: MongoDB, Redis. Для больших и неструктурированных данных.

Раздел 7. DevOps
Инструменты: Docker (контейнеры), Kubernetes (оркестрация), Jenkins (CI/CD).
Задачи: Автоматизация сборки и деплоя приложений.

Раздел 10. Искусственный интеллект (AI)
Machine Learning (ML): Обучение на данных. Инструменты: TensorFlow, PyTorch.
Computer Vision: Распознавание лиц и объектов (YOLO, OpenCV).
NLP: Обработка текста, чат-боты (BERT, GPT).

Раздел 41. Умный дом (IoT)
Платформы: Home Assistant, Apple HomeKit.
Устройства: Умные розетки, лампы Philips Hue, датчики протечки.
Программирование: Arduino, Raspberry Pi.

Раздел 120. Идеи для проектов
1. Telegram-бот с AI на Python (библиотека aiogram).
2. Веб-сайт на React + Node.js.
3. Система умного дома на Arduino.
4. Свой VPN сервер.
